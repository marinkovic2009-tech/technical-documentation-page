# Technical Documentation Page

A responsive technical documentation page built with HTML and CSS as part of the freeCodeCamp Responsive Web Design certification.

## Features
- Fixed side navigation
- Responsive layout for desktop and mobile
- Semantic HTML structure
- Smooth scrolling between sections

## Technologies
- HTML5
- CSS3

## Project Context
This project was created to practice building structured documentation pages with clear navigation and responsive design principles.
